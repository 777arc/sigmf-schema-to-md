The `global` object consists of key/value pairs that provide information applicable to the entire Dataset.
It contains the information that is minimally necessary to open and parse the Dataset file, as well as general information about the Recording itself.

|Field         |Required|Type   |Default|Description                                                                                                                                                                                                                                                                                                                 |
|--------------|--------|-------|-------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|author        |Optional|string |       |A text identifier for the author potentially including name, handle, email, and/or other ID like Amateur Call Sign.                                                                                                                                                                                                         |
|collection    |Optional|string |       |The base filename of a `collection` with which this Recording is associated.                                                                                                                                                                                                                                                |
|dataset       |Optional|string |       |The full filename of the Dataset file this Metadata file describes.                                                                                                                                                                                                                                                         |
|data_doi      |Optional|string |       |The registered DOI (ISO 26324) for a Recording's Dataset file.                                                                                                                                                                                                                                                              |
|datatype      |Required|string |cf32_le|The SigMF Dataset format of the stored samples in the Dataset file.                                                                                                                                                                                                                                                         |
|description   |Optional|string |       |A text description of the SigMF Recording.                                                                                                                                                                                                                                                                                  |
|hw            |Optional|string |       |A text description of the hardware used to make the Recording.                                                                                                                                                                                                                                                              |
|license       |Optional|string |       |A URL for the license document under which the Recording is offered. (RFC 3986)                                                                                                                                                                                                                                             |
|metadata_only |Optional|boolean|       |Indicates the Metadata file is intentionally distributed without the Dataset.                                                                                                                                                                                                                                               |
|meta_doi      |Optional|string |       |The registered DOI (ISO 26324) for a Recording's Metadata file.                                                                                                                                                                                                                                                             |

|recorder      |Optional|string |       |The name of the software used to make this SigMF Recording.                                                                                                                                                                                                                                                                 |
|sample_rate   |Optional|number |       |The sample rate of the signal in samples per second.                                                                                                                                                                                                                                                                        |

Extension Objects MUST contain the three key/value pairs defined below, and MUST NOT contain any other fields.                                                                                            |

The `captures` value is an array of capture segment objects that describe the parameters of the signal capture. It MUST be sorted by the value of each capture segment's `core:sample_start` key, ascending.

|Field       |Required|Type   |Default|Description                                                                                |
|------------|--------|-------|-------|-------------------------------------------------------------------------------------------|
|datetime    |Optional|string |       |An ISO-8601 string indicating the timestamp of the sample index specified by sample_start. |
|frequency   |Optional|number |       |The center frequency of the signal in Hz.                                                  |
|global_index|Optional|integer|       |The index of the sample referenced by `sample_start` relative to an original sample stream.|
|header_bytes|Optional|integer|       |The number of bytes preceding a chunk of samples that are not sample data, used for NCDs.  |
|sample_start|Required|integer|0      |Index of first sample of this chunk.                                                       |

The `annotations` value is an array of annotation segment objects that describe anything regarding the signal data not part of the Captures and Global objects. It MUST be sorted by the value of each Annotation Segment's `core:sample_start` key, ascending.

|Field          |Required|Type   |Default|Description                                                                      |
|---------------|--------|-------|-------|---------------------------------------------------------------------------------|
|comment        |Optional|string |       |A human-readable comment.                                                        |
|freq_lower_edge|Optional|number |       |The frequency (Hz) of the lower edge of the feature described by this annotation.|
|freq_upper_edge|Optional|number |       |The frequency (Hz) of the upper edge of the feature described by this annotation.|
|generator      |Optional|string |       |Human-readable name of the entity that created this annotation.                  |
|label          |Optional|string |       |A short form human/machine-readable label for the annotation.                    |
|sample_count   |Required|integer|       |The number of samples that this Segment applies to.                              |
|sample_start   |Required|integer|0      |The sample index at which this Segment takes effect.                             |
|uuid           |Optional|string |       |RFC-4122 unique identifier.                                                      |
